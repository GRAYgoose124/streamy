# About Streamy

Streamy is a Python library for a basic asyncio event stream implementation.

## Features

- [x] Basic Event Stream
  - [x] EventStream
  - [x] Base Event
  - [x] Base Publisher
  - [x] Base Subscriber 

## Installation

    git clone https://github.com/GRAYgoose124/streamy.git
    cd streamy
    poetry install 

## Usage

    streamy
